05 — Usage Rules
Do & Don't
✓ Do
Use the mark on dark (#101617) or forest (#163628) backgrounds
Maintain clear space of at least 1× the mark height around the logo
Use Silver (#cecece) for all body text on dark backgrounds
Use the green loop accent (#3a9e6a) for CTAs, highlights, and the "2" in the wordmark
Use SVG files for all digital use — they scale infinitely
Use the reversed (dark on light) version only on white/light grey backgrounds
✗ Don't
Never stretch or distort the mark — use uniform scale only
Never use the mark on busy photo backgrounds without a dark overlay
Never change the colour of the loop accent to anything other than #3a9e6a
Never use the wordmark without the correct font weights (700 for "dev" and "c", 700 green for "2")
Never add drop shadows, glows, or outlines to the mark
Never recreate the mark in a different typeface or style
